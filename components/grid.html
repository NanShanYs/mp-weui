# Grid

九宫格

# 引入

import MpGrid from 'mp-weui/packages/grid'

export default {
  components: {
    MpGrid
  }
}

提示

必须与 GridGroup 配合使用。

# 示例

<mp-grid-group>
  <mp-grid
    icon-src="/static/images/icon_tabbar.png"
    label="Grid"
  />
</mp-grid-group>

特别说明

由于 mpvue 暂不支持组件嵌套，所以上述方式暂不适用，具体使用请参考示例代码。

# Props

# href

  • Type: string

# iconSrc

  • Type: string

图标链接。

# label

  • Type: string

说明文字。

  • Parameter: $event
